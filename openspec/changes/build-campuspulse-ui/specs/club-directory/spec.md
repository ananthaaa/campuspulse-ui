## ADDED Requirements

### Requirement: club-directory-grid
The system SHALL display a showcase grid of campus clubs represented by their logos.

#### Scenario: Displaying club catalog
- **WHEN** the user visits the Club Directory page
- **THEN** the system SHALL display the grid of clubs.

### Requirement: club-profile-detail
The system SHALL display a profile page for each club including club information, a grid of club members, and a list of upcoming events hosted by the club.

#### Scenario: Viewing club profile
- **WHEN** the user clicks on a club logo card
- **THEN** the system SHALL navigate to the Club Profile showing members and upcoming events.
