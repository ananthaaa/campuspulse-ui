## ADDED Requirements

### Requirement: Admin Venue Upload
The system SHALL provide an interface for administrators to upload SVG floor plans for venues.

#### Scenario: Uploading a floor plan
- **WHEN** an admin accesses the Venue Setup page
- **THEN** an image upload zone is displayed for the SVG file.

### Requirement: Waypoint Editor
The system SHALL allow administrators to define and order navigational waypoints for a venue.

#### Scenario: Adding a waypoint
- **WHEN** an admin uses the waypoint editor
- **THEN** they can add step-by-step text instructions that map to the uploaded floor plan.
