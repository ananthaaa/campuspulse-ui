## ADDED Requirements

### Requirement: Student Login UI
The system SHALL present a styled login form for student authentication.

#### Scenario: Rendering the login form
- **WHEN** an unauthenticated user views the profile page
- **THEN** the sleek dark-themed login form is displayed.

### Requirement: Student Profile Dashboard
The system SHALL display the student's RSVP history and a role toggle switch.

#### Scenario: Viewing the authenticated profile
- **WHEN** an authenticated student views their profile
- **THEN** they see their avatar, a list of RSVP'd events, and a "Switch to Admin" toggle.
