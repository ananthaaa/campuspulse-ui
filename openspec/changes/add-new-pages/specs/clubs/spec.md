## ADDED Requirements

### Requirement: Club Directory Listing
The system SHALL display a grid of club cards representing campus organizations.

#### Scenario: Viewing the directory
- **WHEN** the user navigates to the Club Directory
- **THEN** they see a grid of club logos, names, and short descriptions.

### Requirement: Club Profile View
The system SHALL display detailed information about a specific club upon selection.

#### Scenario: Viewing a club profile
- **WHEN** the user clicks on a club card in the directory
- **THEN** they are navigated to the Club Profile showing the club's upcoming events and member roster.
