## ADDED Requirements

### Requirement: Mock Login Flow
The system SHALL provide a realistic UI for user authentication simulation.

#### Scenario: Student Login
- **WHEN** a user clicks "Login" and selects the Student role
- **THEN** the system simulates a login and redirects them to the Student Discover page

#### Scenario: Admin Login
- **WHEN** a user clicks "Login" and selects the Admin role
- **THEN** the system simulates a login and redirects them to the Admin Dashboard
